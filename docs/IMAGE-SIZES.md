# Agnes AI 图像尺寸参考

来源：https://platform.agnes-ai.com

## API 调用格式

```javascript
{
  "model": "agnes-image-2.1-flash",  // 或其他模型
  "prompt": "...",
  "size": "1K" | "2K" | "3K" | "4K",
  "ratio": "1:1" | "3:4" | "4:3" | "16:9" | "9:16" | "2:3" | "3:2" | "21:9",
  "extra_body": { "response_format": "url" }
}
```

## 实际项目推荐（sourdough-calculator）

| 用途 | API 参数 | 像素 | 备注 |
|------|---------|------|------|
| 博客 hero 图 | 1K 16:9 | 1312x736 | 显示 ~768px，1K 视网膜足够 |
| 首页 hero 区 | 2K 21:9 | 3136x1344 | 全屏宽，2K+ 必需 |
| About 页插图 | 1K 4:3 | 1152x864 | 中等方形 |
| OG 分享卡 | 1K 16:9（裁 1.91:1） | 1312x736 | 切两侧变 1200x630 |
| 缩略图 | 1K 1:1 | 1024x1024 | 列表卡片用 |
| 移动端竖图 | 1K 9:16 | 736x1312 | Story 类 |
| 产品横图 | 1K 3:2 | 1248x832 | 印刷风 |

## 选择原则

- **1K**：移动端友好，省流量，文件小
- **2K**：全屏或 hero 大图，Retina 友好
- **3K/4K**：仅当真正需要大幅打印/放大用

## 注意事项

1. 1920x1080 / 2560x1440 是**显示器**分辨率，不是 Agnes 原生输出
2. 不支持的精确 size×ratio 会**自动标准化**到最近的标准档
3. PNG 文件大小因内容复杂度而异（同样 1312x736 可能是 600KB 或 1.2MB）
4. WebP 格式比 PNG 小约 30-50%（如需优化）
